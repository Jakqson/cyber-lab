Learn basic Git: push, commit, clone, branching
###### Clone
Used to copy a project from GitHub to your computer.
> git clone https://github.com/jakqson/cyber-lab.git
###### Commit
> git add .  
> git commit -m "Added login feature"

`git add .`: stages all changes
`git commit -m` : saves them with a message

###### Push 
upload to GitHub
> git push origin main

origin : your GitHub repo
main : branch name

###### Branching
Branches let you work on new features without affecting the main code.

Create a branch:
> git branch feature-login

Switch to it:
> git checkout feature-login
